Projects

Intelligent Animal Repelling System

The project addressed animal migration toward human settlements. It would detect the ungulates and based on their sensitive frequency range, generate a tailored Ultrasonic frequency that would harmlessly repel the animal in the subject.

Disclaimer: No animals were harmed during the making or testing phase of this project. The project working was proven with the help of a lcd display and its effectiveness was proven through the IEEE research papers which agreed on the effect of this project would bring about.

Autonomous Maze Solving Robot

The project aimed to create an autonomous maze-solving robot, using left-wall following algorithm. Ultrasonic sensors detect obstacles in front of the robot and the maze wall to the left. With Arduino Uno as the microcontroller, along with voltage regulators and motor controllers formed the robot's hardware. Arduino IDE was used to program the logic and the programmed navigation logic enabled autonomous maze navigation.
